Citizen Support and Civic Issue Management System
--------------------------------------------------

This is a Python Django-based web application that allows citizens to report and track civic issues such as damaged roads, power failures, and water leakage.

Modules:
--------
1. Citizen Module
   - Register, log in, submit complaints
   - View and track status
   - Provide feedback

2. Authority Module
   - View assigned complaints
   - Update complaint status
   - View feedback

3. Admin Module
   - Manage users and departments
   - Assign complaints
   - View reports and statistics

4. Notification Module
   - Sends automatic email updates to users

Technologies Used:
------------------
- Backend: Python Django
- Frontend: HTML, CSS, JavaScript
- Database: SQLite
- Email Notification: SMTP (Django)
